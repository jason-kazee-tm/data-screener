# Invoice Data Quality Management System

This repository contains the Invoice Data Quality Management System which is structured to support both a backend and a frontend application.

## Directory Structure
- `src/`
  - `backend/` - Python backend for data management.
  - `frontend/` - Node.js and TypeScript frontend for user interaction.
  - `config/` - Configuration files for the application.
- `docs/` - Documentation related to the project.
- `tests/` - Test cases and testing frameworks.

## Configuration Files
- `README.md` - Project overview.
- `setup.py` - Setup file for Python packages.
- `package.json` - Node.js package management.
- `requirements.txt` - Python dependencies.
- `tsconfig.json` - TypeScript configuration.
- `docker-compose.yml` - Docker Compose configuration for setting up the environment.

## Getting Started
1. Clone the repository.
2. Navigate to the directory.
3. Install the necessary dependencies for both backend and frontend.
4. Configuration settings can be adjusted in the `src/config` directory.
